# Simple Deployment Guide (Using Existing Database)

Since you already have a free database on Render, follow these steps:

---

## Step 1: Deploy Using Blueprint (Without Database)

1. Go to https://dashboard.render.com/
2. Click **"New +"** â†’ **"Blueprint"**
3. Connect your GitHub repo: `VeeCC-T/ShieldHer`
4. Click **"Apply"**

This will create:
- âœ… Backend web service
- âœ… Frontend static site

---

## Step 2: Connect Your Existing Database

### Find Your Database URL:
1. Go to your existing database in Render dashboard
2. Click on it
3. Scroll down to **"Connections"**
4. Copy the **"Internal Database URL"** (starts with `postgresql://`)

### Add to Backend:
1. Go to **shieldher-backend** service
2. Click **"Environment"** tab
3. Find `DATABASE_URL` variable
4. Paste your database URL
5. Click **"Save Changes"**

---

## Step 3: Configure Environment Variables

### Backend Environment Variables:
Go to **shieldher-backend** â†’ **Environment** tab and verify/add:

```
DJANGO_SETTINGS_MODULE=config.settings.production
SECRET_KEY=<auto-generated>
DATABASE_URL=<your-existing-database-url>
ALLOWED_HOSTS=.onrender.com
DEBUG=False
CORS_ALLOWED_ORIGINS=https://shieldher-frontend.onrender.com
```

### Frontend Environment Variables:
Go to **shieldher-frontend** â†’ **Environment** tab and add:

```
VITE_API_URL=https://shieldher-backend.onrender.com/api
```

---

## Step 4: Run Database Migrations

1. Go to **shieldher-backend** service
2. Click **"Shell"** tab
3. Run:
   ```bash
   python manage.py migrate
   python manage.py createsuperuser
   ```

---

## Step 5: Get Your Live URLs

Your app will be live at:
- **Frontend**: `https://shieldher-frontend.onrender.com`
- **Backend**: `https://shieldher-backend.onrender.com/api`
- **Admin**: `https://shieldher-backend.onrender.com/admin`

## Alternative: Manual Deployment (If Blueprint Still Fails)

### Deploy Backend Manually:

1. **New +** â†’ **Web Service**
2. Connect GitHub: `VeeCC-T/ShieldHer`
3. Configure:
   - **Name**: `shieldher-backend`
   - **Environment**: Docker
   - **Dockerfile Path**: `backend/Dockerfile`
   - **Docker Context**: `./backend`
   - **Plan**: Free
4. Add environment variables (see Step 3)
5. Click **"Create Web Service"**

### Deploy Frontend Manually:

1. **New +** â†’ **Static Site**
2. Connect GitHub: `VeeCC-T/ShieldHer`
3. Configure:
   - **Name**: `shieldher-frontend`
   - **Root Directory**: `frontend`
   - **Build Command**: `npm install && npm run build`
   - **Publish Directory**: `dist`
4. Add environment variable: `VITE_API_URL=https://shieldher-backend.onrender.com/api`
5. Click **"Create Static Site"**
